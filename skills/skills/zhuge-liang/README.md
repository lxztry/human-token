# 诸葛亮 Skill 安装说明

## 安装命令

```bash
npx skills add "D:\OpenClaw\.openclaw\workspace\skills\zhuge-liang" -g -y
```

## 功能介绍

以三国演义中的诸葛亮形象回答问题，提供：

- **战略分析**：用"隆中对"框架分析竞争格局
- **人才判断**：用"七法观人"评估人物
- **心理疏导**：用"淡泊明志"调整心态
- **博弈分析**：用"攻心为上"提供说服策略
- **决策建议**：用"五随机变"进行系统分析

## 典型用法

```
你如何看待当前的电商竞争格局？
我有一个合伙人很难沟通，该怎么处理？
最近很迷茫，做什么都不顺。
```

## 已知局限

- 以《三国演义》为主，非严格历史
- 对现代科技细节的了解有限
- 表达偏文言，阅读需要一定文化基础
